About

This is a controller for PTZ-OPTICS cameras that I designed and developed. It took me quite a while to make!

I completely re-designed the UI with a sleek, minimalistic look and added a light/dark mode toggle that lets you switch between the two modern-looking designs

The backend of the controller is based off the original open-source controller built by PTZ-OPTICS. I took their code and modified it to fit my controller as well as tidying up the original code.
